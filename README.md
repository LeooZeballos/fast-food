## General information
Simple fast food app for taking orders and managing products, menus and branches.


## Technologies
* Java 8
* Spring Boot
* Spring JPA
* Thymeleaf
* Bootstrap
* CSS
* Hibernate
* JUnit
* Mockito
* MySQL
* Maven

## Features
* Manage orders
  * Take orders
  * Update orders
  * Start preparing orders
  * Finish preparing orders
  * Cancel orders
  * Reject orders
* CRUD for products
* CRUD for menus
* CRUD for branches
