# Designer

## Overview

This skill equips Gemini CLI with UI/UX design expertise for the FastFood project. It focuses on making the application look professional, modern, and user-friendly through visual hierarchy, consistent spacing, and interactive elements.

## Guidelines

### 1. Visual Aesthetics
- **Spacing**: Use consistent margins and padding to create breathing room in the layout.
- **Color Palette**: Maintain the project's color theme (red, orange, yellow hues) for a "fast food" feel.
- **Typography**: Use clear, readable fonts with distinct weights for headings and body text.

### 2. UI Principles
- **Visual Hierarchy**: Guide the user's eye to the most important actions (e.g., "Add to Order", "Submit").
- **Interactive Feedback**: Ensure buttons and links have hover states and clear visual changes when clicked.
- **Modern Polish**: Use subtle shadows, rounded corners, and smooth transitions.

### 3. Layout and Organization
- **Responsiveness**: Ensure the design works well across different screen sizes.
- **Consistency**: Keep buttons, forms, and lists consistent throughout the application.

## Workflow

1. **UX Audit**: Identify areas in the current UI that feel outdated or hard to use.
2. **Design**: Propose a layout and styling that improves the user's journey.
3. **Implementation**: Apply the styling changes and verify the visual outcome.
